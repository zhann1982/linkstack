HTML - Hypertext markup language

HTML (HyperText Markup Language — «язык гипертекстовой разметки») — самый базовый строительный блок Веба. Он определяет содержание и структуру веб-контента. Другие технологии, помимо HTML, обычно используются для описания внешнего вида/представления (CSS) или функциональности/поведения (JavaScript) веб-страницы.

Main Links

HTML | MDN web docs - Руководство для начинающих от Mozilla.org. На этой странице расположенные главные ссылки на темы связанные с разметкой страницы.

Справочник по HTML от htmlbook.ru - Удобный справочник по тэгам в алфавитном порядке. Есть ссылка на аттрибуты тэгов. Отдельно выделены тэги, которые были добавлены в стандарте HTML5.

HTML Tutorial - W3Schools - Обширный туториал по разметке на английском языке с интерактивными примерами "Try it Yourself".Наверное это самый подробный и полный истончик знании по html.

HTML — Википедия - Основная страничка на русском языке на портале Википедия. Любой может принять участие в редактировании и дополнении этой страницы.

HTML5 book - Еще один русскоязычный портал с подробной информацией по тэгам и не только. На этой странице акцент сделан на стандарт html5

HTML standart page from W3C and WHATWG - Как вы уже знаете? есть организации которые занимаются страндартизацией веб-разработки. Недавно две такие организации объединили усилия на создании единого портала по стандартам html. Данный портал содержит самые свежие изменения в стандарте. Также можно почитать информацию по грядущим изменениям.

DOM standart page from W3C and WHATWG - Единая страница по спецификации DOM - Document Object Model. Вы наверняка уже знакомы с этой темой. Говоря упрощенно, данная технология позволяет манипулировать элементами на веб-странице с помощью языка программирования. Раньше для этого использовали только Javascript. Но сейчас у разработчиков выбор практически не ограничен.
